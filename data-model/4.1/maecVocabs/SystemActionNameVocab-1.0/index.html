SystemActionNameVocab-1.0MAEC Vocabularies Schema


API support

The SystemActionNameVocab is the default MAEC vocabulary for system action names, captured via the ActionType/Name element in CybOX Core. For system action names, it should be used in place of the CybOX ActionNameVocab-1.0.
