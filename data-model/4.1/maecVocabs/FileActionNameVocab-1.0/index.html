FileActionNameVocab-1.0MAEC Vocabularies Schema


API support

The FileActionNameVocab is the default MAEC vocabulary for file action names, captured via the ActionType/Name element in CybOX Core. For file action names, it should be used in place of the CybOX ActionNameVocab-1.0. Deprecated as of MAEC 4.1.
