SynchronizationActionNameVocab-1.0MAEC Vocabularies Schema


API support

The SynchronizationActionNameVocab is the default MAEC vocabulary for synchronization action names, captured via the ActionType/Name element in CybOX Core. For synchronization action names, it should be used in place of the CybOX ActionNameVocab-1.0.
